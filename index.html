Program overview!

"Make Me A Coder" is a digital empowerment initiative aimed at training youth and
aspiring developers in full-stack web development. The project focuses on equipping
participants with practical skills in programming, database management, and web
technologies to prepare them for opportunities in the tech industry.

Technical Stack

  • Languages: HTML, CSS, JavaScript, Python, SQL
  • Frameworks: Django, Flask
  • Operating System: Kali Linux OS
  • Tools: Visual Studio Code, Git/GitHub, MySQL/SQLite, Chrome/Firefox
  • Colaboration Platform: Visual Studio Code, Git/GitHub, MySQL/SQLite, Chrome/Firefox

Infrastructure & Resources

  1. Laptops (Intel i5 or higher, 8GB RAM, 250 GB SSD minimum)
  2. Software: Kali Linux OX, Python, MySQL, Git, Microsoft Teams, MS Office
  3. Accessories:
    • USB Drives (8GB minimum) - for installing and booting Kali Linux
    • Projector or display screen for instructional sessions
  4. Connectivity: Reliable internet connection for research and collaboration

Expected outcome

Students will be able to design, code, and deploy complete web applications,
manage databases, and collaborate effectively using Microsoft Teams and GitHub.
